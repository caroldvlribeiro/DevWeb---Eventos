
# 🎉 Sistema de Gerenciamento de Eventos

Projeto desenvolvido em Laravel com o objetivo de realizar o gerenciamento de eventos, permitindo o cadastro, edição, visualização e exclusão de eventos de forma simples e organizada.

## 📋 Sobre o Projeto

O Sistema de Gerenciamento de Eventos foi criado para facilitar o controle de eventos, centralizando informações como título, descrição, data, valor e quantidade de vagas disponíveis.

O projeto foi desenvolvido utilizando o padrão MVC (Model-View-Controller), proporcionando uma estrutura organizada, escalável e de fácil manutenção.

## 🚀 Funcionalidades

- Cadastro de eventos
- Listagem de eventos
- Visualização de detalhes dos eventos
- Edição de eventos cadastrados
- Exclusão de eventos
- Validação de formulários
- Interface responsiva utilizando Bootstrap
- Integração com banco de dados

## 🛠️ Tecnologias Utilizadas

- Laravel
- PHP
- Blade Templates
- Bootstrap
- MySQL
- HTML5
- CSS3
- Git e GitHub

## 📂 Estrutura do Projeto

O sistema segue a arquitetura MVC:

- **Model:** Responsável pela manipulação dos dados.
- **View:** Interface apresentada ao usuário.
- **Controller:** Responsável pelas regras de negócio e comunicação entre Model e View.
